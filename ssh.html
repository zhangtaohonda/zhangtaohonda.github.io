## ssh 除了能够实现常用的安全登陆之外，还有一个端口转发的功能，借助ssh通道，你可以穿透网络限制访问到你想访问的内容 -D 动态转发 -L 本地转发 -R 远程转发 -N 不执行shell -f 将ssh转到后台运行，一般用于端口转发时使用 -g 允许打开的端口让远程主机运行 -C 压缩数据传输 -T 不分配tty，只做代理用 #### 端口转发类型 1.动态转发 ssh -C -f -N -g -D listen_port user@Tunnel_Host 2.本地端口转发 ssh -C -f -N -g -L listen_port:DST_Host:DST_port user@Tunnel_Host 3.远程端口转发 ssh -C -f -N -g -R listen_port:DST_Host:DST_port user@Tunnel_Host
